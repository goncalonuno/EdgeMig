NodeDiscovery.cone_determination(client, coor_index)¶

The cone_determination function determines the cone that the client will have when running the trip.

Parameters

  • client (Object of class clientclass) – The client object that has all the information about the client of that trip

  • coor_index (Integer) – Index of the trips dataframe

Returns

Returns 1 in success

Return type

Integer

NodeDiscovery.get_client_source(client, dfstations)¶

The get_client_source function finds the closest edge server to the user by comparing the distance between the client coordinates and the coordinates of the edge server station. This is used to find the first edge server as it starts the trip

Parameters

  • client (Object of class clientclass) – The client object that has all the information about the client of that trip

  • dfstations (Pandas dataframe) – Dataframe with the edge server stations

Returns

Returns 1 in success

Return type

Integer

NodeDiscovery.node_search(client, coor_index, dfstations, user_def)¶

The node_search function suggest a possible destination based on the client’s heading through the use of a search cone and by taking into account the latency or distance that possible destination.

Parameters

  • client (Object of class clientclass) – The client object that has all the information about the client of that trip

  • coor_index (Integer) – Index of the trips dataframe

  • dfstations (Pandas dataframe) – Dataframe with the edge server stations

  • user_def (Object of the class UserDef) – User definitions

Returns

Returns 1 it finds a possible destination or it returns -1 if it can’t find a possible destination

Return type

Integer

NodeDiscovery.node_search_dist(client_coor, st_lat_lon, server_origin_coor)¶

The node_search_dist function checks if the distance to the target edge server is within range when comparing it to the distance that the client has to the source.

Parameters

  • client_coor (Tuple of floats) – The client coordinates

  • st_lat_lon (Tuple of floats) – The target station coordinates

  • server_origin_coor (Tuple of floats) – The source station coordinates

Returns

Returns 1 if the destination is in range or it returns -1 if if the destination is not in range

Return type

Integer

NodeDiscovery.remove_cluster(dfstations)¶

The remove_cluster function removes clusters of edge servers. This is to prevent migrations between edge servers that are to close together and mitigate uncessary migrations.

Parameters

dfstations (Pandas dataframe) – Dataframe with the edge server stations

Returns

Returns the filtered edge server dataframe

Return type

Pandas Dataframe
