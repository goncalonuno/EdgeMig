dynamic_plot module¶

dynamic_plot.Animate(i, dftrips, dfstations, lock)¶

The Animate function is responsible for plotting the data that is being saved in real time when running the simulation

Parameters

  • i (Integer) – Frame number

  • dftrips (Pandas dataframe) – Dataframe with the trips

  • dfstations (Pandas dataframe) – Dataframe with the edge server stations

  • lock (Multiprocessing lock) – Multiprocessing lock for accessing memory

Returns

Returns 1 in case of success

Return type

Integer

dynamic_plot.df_dynamic_plot(client, dfnode, coor_index, ret_node, lock)¶

The df_dynamic_plot function is responsible adding the newly generated data to the dataframe that will be used by the dynamic plot.

Parameters

  • client (Object of class clientclass) – The client object that has all the information about the client of that trip

  • dfnode (Pandas dataframe) – Dataframe used for saving the data required by the dynamic plot

  • coor_index (Integer) – Index of the coordinates from the trips dataframe

  • ret_node (Integer) – Defines if the given coordinate has no possible migration destination

  • lock (Multiprocessing lock) – Multiprocessing lock for accessing memory

Returns

Returns a dataframe with the new data added at the end

Return type

Pandas Dataframe

dynamic_plot.heading_to_coordinate(o_lat, o_lon, angle, dist)¶

The heading_to_coordinate function is responsible for converting a heading value into a coordinate

Parameters

  • o_lat (Float) – Latitude

  • o_lon (Float) – Longitude

  • angle (Float) – Heading angle

  • dist (Float) – Distance from the provided coordinate

Returns

Returns a list with the converted latitude and longitude

Return type

List

dynamic_plot.init()¶

The init function initiates the data that will be plotted dynamically

Returns

Returns the line where the data will be plotted

Return type

List of Line2D

dynamic_plot.plot_process(dftrips, dfstations, lock)¶

The plot_process function is responsible for starting the plotting procedure that it will then be animated.

Parameters

  • dftrips (Pandas dataframe) – Dataframe with the trips

  • dfstations (Pandas dataframe) – Dataframe with the edge server stations

  • lock (Multiprocessing lock) – Multiprocessing lock for accessing memory

Returns

Returns 1 in case of success

Return type

Integer
